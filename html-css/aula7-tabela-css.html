Cada navegador tem a sua apresentação por padrão e esse comportamento individualizado pode gerar conflitos de layout diante do seu projeto web. O reset CSS reseta tags que tenham valores iniciais, o que ocasiona pré formatação. A técnica é uma forma de suavizar as diferenças e padronizar a estiização, sobrepondo a formatação original do browser com uma folha de estilo.
